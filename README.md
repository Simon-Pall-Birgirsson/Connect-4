# Connect-4
VESM1VS05AU verkefni-6 - Connect 4

Höfundar: Símon Páll Birgirsson og Jökull Sverrir heiðarsson
Tveir leikmennir skiptast á að setja X og O á LED 9x6 borðið. Þeir nota takkanna á kassan til að fara til vinstri, hægri og að sleppa. Ef það er 4,5 eða 2x2 kassi, X eða O í röð fær leikmaður X eða O 10,25,15 stig.
